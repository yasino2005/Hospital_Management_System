# Hospital_Management_System
هدف هذا المشروع إنشاء نظام إدارة مشفى يساعد على تنظيم وإدارة العمليات الأساسية داخل المشفى مثل تسجيل المرضى، إدارة المواعيد، متابعة الأطباء والأقسام، وتوثيق المعلومات الطبية. يركز المشروع على العمل الجماعي باستخدام أدوات حديثة لمحاكاة بيئة العمل الحقيقية في سوق العمل وتحسين مهارات إدارة المشاريع البرمجية
